# Digital Well-Being Analytics Toolkit (DWAT)
**An Open-Source Framework for Algorithmic Auditing in Public Education**

## 🚀 Project Overview
The **Digital Well-Being Analytics Toolkit (DWAT)** is the flagship initiative of the *Open-Source Digital Well-Being Lab*. It is designed to empower schools, libraries, and youth-serving NGOs with the technical capability to audit, monitor, and mitigate the impact of algorithmic feedback loops on minors.

## 🛠 Key Features
- **Privacy-First Design:** Fully compliant with FERPA and COPPA through built-in Differential Privacy modules.
- **Causal Auditing:** Advanced statistical engines to distinguish between organic usage and algorithmically-driven addictive patterns.
- **Transparent Benchmarking:** Open standards for "Wholesome Content" classification using scalable LLM knowledge distillation.

## 📂 Repository Structure
- `/docs`: Full Technical Whitepaper and Architectural Blueprints.
- `/core`: Initial Python framework for data ingestion and privacy-preserving de-identification.
- `/models`: Schemas for causal inference and risk-indicator tracking.
